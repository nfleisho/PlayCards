    This page simulates dealing bridge hands.
  • Deal Hand will take you to a page showing one deal
  • More Info.. will take you to a theoretical discussion of probabilities.
  • Number of Shuffles sets the number of times the deck is shuffled before each deal. One or two times is probably sufficient. (Can you guess what the graph will look like with zero shuffles?) The graph will redraw after return or tab.
  • Number of Deals sets the number of times the deck is dealt during data collection. The number of hands sampled is four times the number of deals. The more deals you enter, the longer it will take for the graph to appear. The graph will redraw after return or tab.
